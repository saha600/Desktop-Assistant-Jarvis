
# Desktop Assistant (Jarvis)

Replace `YOUR_INTERN_ID` with your CodTech Intern ID.

## Features
- Time & Date
- Google & YouTube
- Google Search
- Wikipedia (opens browser)
- Play YouTube songs
- Jokes
- Notepad
- Calculator
- Paint
- Command Prompt
- File Explorer
- Screenshot

## Install

```bash
python -m pip install -r requirements.txt
```

Run:

```bash
python jarvis.py
```
