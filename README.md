## Project Description

Desktop Assistant (Jarvis) is a Python-based virtual assistant developed using Python. It performs various desktop automation tasks such as opening applications, searching Google, opening websites, playing YouTube songs, taking screenshots, and more through simple text commands.

---

## Features

- Greeting based on current time
- Text-to-Speech
- Display current Time
- Display current Date
- Open Google
- Open YouTube
- Search Google
- Open Wikipedia pages
- Play YouTube songs
- Tell random jokes
- Open Notepad
- Open Calculator
- Open Paint
- Open Command Prompt
- Open File Explorer
- Take Screenshots
- Exit the application

## Installation

Install the required libraries:

```bash

```

Run the project:

```bash
python jarvis.py
```
